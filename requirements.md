
贴膜门店管理系统  
整体需求说明书（V1.0）

适用场景：汽车贴膜连锁品牌（含总部 + 多家直营/加盟门店）  
系统目标：实现“客户体验数字化、施工流程标准化、门店运营可视化、总部管控一体化”

一、系统角色与权限体系
角色   所属层级   主要职责
总部管理员   平台级（Tenant Admin）   管理所有门店、查看全网数据、配置全局规则、审核品牌物料

门店店长   租户级（Store Manager）   管理本店员工、服务、预约、库存、营销活动

门店员工（技师/前台/销售）   执行层   接收工单、施工记录、客户服务

车主用户   C端用户   预约服务、查看进度、评价售后

✅ 多租户隔离原则：  
- 每家门店为一个独立租户（Tenant），数据完全隔离  
- 总部可跨租户查看汇总数据及明细（需权限控制）  
- 全局配置（如膜品牌库、SOP模板）由总部维护，门店可继承或局部覆盖

二、核心功能模块（按端划分）

（一）用户端（C端小程序/H5）

1. 账户与车辆管理
- 微信一键登录，绑定多辆车
- 查看历史施工记录、质保信息

2. 服务浏览与智能推荐
- 按车型自动匹配贴膜方案
- 展示价格、施工时长、膜材参数、效果图

3. 在线预约
- 选择门店 → 日期/时段 → 服务项目 → 技师（可选）
- 支持备注特殊需求（如“需洗车后施工”）

4. 订单与施工追踪
- 实时查看施工进度（阶段化状态）
- 查看施工过程照片/视频（由员工上传）
- 下载电子工单 & 质保卡（带防伪二维码）

5. 营销互动
- 领取优惠券（通用 / 店长专属 / 裂变券）
- 老带新邀请（生成专属海报，双方得奖励）
- 生日礼包自动发放

6. 售后与评价
- 在线报修（上传问题照片）
- 服务完成后评价技师与门店

7. 消息通知
- 微信服务通知：预约确认、到店提醒、完工通知、质保到期等

（二）员工端（一线执行App/小程序）

1. 今日任务看板
- 按时间排序展示当日预约客户
- 显示：车牌、车型、服务项目、预计到店时间

2. 工单执行
- 一键签到（客户到店）
- 分阶段更新施工状态（清洗 → 裁膜 → 铺膜 → 收边 → 质检）
- 上传各阶段施工照片（自动加水印：时间+车牌+员工名）

3. 物料与质检
- 扫码登记所用膜卷编号（用于溯源与库存扣减）
- 标记异常（如车身旧伤、客户临时加项）

4. 客户协作
- 快捷呼叫前台/销售协助
- 引导客户评价：“稍后请点亮五星好评哦！”

5. 个人中心
- 查看本月完成单数、提成明细
- 查看个人星级（基于好评率、返工率）

6. 知识赋能
- 查阅标准施工视频（按车型/膜类型）
- 新品膜材特性速查

（三）商家端（门店管理后台）

1. 门店基础设置
- 门店信息（名称、地址、营业时间）
- 可预约时间段配置
- 打印模板 & 工单尾注设置

2. 客户管理
- 车主档案管理（含车辆绑定）
- 沉睡客户唤醒、生日关怀计划
- 客户标签（如“高端车主”“隐形车衣偏好”）

3. 服务与商品
- 服务项目配置（部位、指导价、施工SOP）
- 膜材商品库管理（品牌、型号、成本价、零售价）
- 库存预警（低库存自动提醒）

4. 预约与工单
- 预约列表管理（支持手动调整技师）
- 工单打印 / PDF导出 / 图片分享

5. 员工与提成
- 员工档案管理（角色、岗位、手机号）
- 提成规则配置（按项目固定金额 / 按比例 / 阶梯奖励）

6. 营销工具
- 发放优惠券（定向/通用）
- 配置抽奖活动、复购返利规则

7. 数据报表
- 日/周/月营业额、单量、客单价
- 员工绩效排行、热门服务TOP榜

（四）总部管理端（多租户管控平台）

仅总部管理员可见

1. 门店管理
- 新增/停用门店（分配租户ID）
- 查看门店列表（状态、负责人、城市）

2. 全局配置中心
- 统一膜品牌库（供所有门店引用）
- 标准施工SOP模板库
- 全局消息模板（如质保条款、免责声明）

3. 跨门店数据看板
- 全网营收趋势、门店排名
- 区域热力图（高消费区域分析）
- 客户复购率、NPS（净推荐值）统计

4. 明细穿透查询
- 可下钻查看任意门店的：
  - 客户订单明细
  - 员工施工记录
  - 优惠券核销情况
  - 售后问题分布

5. 合规与审计
- 操作日志（谁在何时修改了什么）
- 膜材溯源追踪（从总部采购 → 门店入库 → 客户使用）

6. 加盟支持（可选）
- 为加盟店提供标准化运营包（菜单、价格、话术）
- 远程培训课程推送

三、多租户架构设计要点
维度   实现方式
数据隔离   每个门店拥有独立 tenant_id，所有业务表包含该字段，查询自动过滤

权限控制   RBAC + Tenant Scope，总部账号拥有 super_admin 权限，可切换租户上下文

配置继承   总部发布“标准服务包”，门店可启用/微调，但不可删除全局品牌

统一登录   员工/店长通过手机号+验证码登录，系统自动识别所属门店

API网关   所有接口校验 X-Tenant-ID，防止越权访问

四、非功能性需求

- 性能：工单加载 <1s，图片上传支持断点续传  
- 安全：客户隐私数据脱敏，施工照片仅限关联客户与员工可见  
- 兼容性：支持微信小程序 + 企业微信 + Web管理后台  
- 离线支持：员工端在网络不佳时可暂存施工记录，恢复后自动同步  
- 扩展性：预留接口支持未来接入DMS（经销商管理系统）或保险公司合作

五、后续演进建议

1. AI辅助：通过施工照片自动识别气泡、翘边等质量问题  
2. IoT集成：车间温湿度传感器联动施工提醒（某些膜需特定环境）  
3. 供应链打通：膜材库存联动供应商自动补货  
4. 保险合作：贴膜即送“膜损险”，理赔流程嵌入系统
